关于这个项目

这是一个使用现代前端技术栈构建的复杂项目示例。它使用Webpack进行模块打包， 包含ES6模块、SCSS样式预处理、以及现代化的JavaScript功能。

项目特点

模块化架构

使用ES6模块系统，代码组织清晰，易于维护和扩展。

现代CSS

使用SCSS预处理器，支持变量、嵌套、混合等功能。

自动化部署

每次更新代码都会触发 GitHub Actions 自动构建和部署。

响应式设计

适配各种屏幕尺寸，提供一致的用户体验。

交互功能

本项目包含多种交互功能，如动态粒子效果、主题切换、滚动动画等。 点击下面的按钮体验不同功能。

技术说明

这个项目使用了现代前端开发工具链。Webpack作为模块打包器， 将所有资源（JavaScript、CSS、图片等）打包成优化后的静态资源。

源代码结构清晰，包含模块化的JavaScript代码、SCSS样式文件， 以及可扩展的HTML模板。

构建与部署

使用npm脚本可以轻松进行开发、构建和测试。GitHub Actions工作流 会自动处理代码的构建和部署到GitHub Pages的过程。
